# Decision Log

This document records important decisions made during the Thrive AI project. It should explain what was decided and why.

## 2026-05-17

### Decision: Project name changed to Thrive AI

The project name was chosen as **Thrive AI** because it is short, positive, and broad enough to cover many areas of personal growth.

### Decision: Main tagline selected

The chosen tagline is:

**Find your drive. Start to thrive.**

This tagline connects directly to the project's personal meaning: people can only thrive when they discover and use their own drive.

### Decision: Documentation will be updated throughout the project

The project will include ongoing documentation, including progress logs, session notes, and decision records. This will show the effort, planning, and development process behind the project.

### Decision: GitHub repository selected

The GitHub repository for the project is:

https://github.com/MACTEN2/THRIVE

The local project folder will be connected to this repository so the project can be version controlled and shared.

### Decision: First GitHub push completed from local Thrive-AI folder

The local project folder at `Thrive-AI` was initialised as a Git repository, connected to the GitHub remote, and pushed to the `main` branch. This establishes GitHub as the shared version history for the project.
