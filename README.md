# Thrive AI

**Tagline:** Find your drive. Start to thrive.

Thrive AI is a personal AI agent project designed to help people turn real-life challenges into clear, achievable action plans. It supports individuals with goals such as finding a job, improving confidence, building healthier habits, managing studies, planning assignments, and creating positive routines.

The core belief behind the project is simple:

> A person can only thrive when they find their drive.

Thrive AI is not just a chatbot. It is intended to become a practical support system that combines conversation, planning, memory, progress tracking, and safe guidance to help users make meaningful progress in their everyday lives.

## Current Project Status

This project is in the early concept and planning stage. The first development focus is to define the project clearly before moving into features, system design, data storage, real-life impact, and future development.

## Planned Build Stack

- **AI/backend language:** Python
- **Backend framework:** FastAPI
- **Frontend:** TypeScript with React or Next.js
- **Database:** PostgreSQL
- **AI memory/search:** pgvector
- **Deployment:** Docker with a cloud platform such as Render, Fly.io, or AWS

Python is the recommended core language because it is widely used in AI, automation, backend systems, data handling, and agent development. TypeScript is a strong frontend choice because it is highly demanded for modern web applications.
